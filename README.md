# Login

<h1>About</h1>
Secure Login Screen

<h1>Features</h1>
* Face Recognition Enabled <br>
* Fingerprint Recognition Enabled <br>
* Biometric Authentication <br>
* Email / Password Authentication <br>
* Storing credentials through Keychain <br>
* Alerts when credentials are not valid <br>
